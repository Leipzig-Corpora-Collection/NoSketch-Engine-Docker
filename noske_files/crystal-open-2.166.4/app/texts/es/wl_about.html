La herramienta de listas de palabras genera listas de frecuencias de varios tipos:
  • sustantivos, verbos, adjetivos y otras partes de la oración
  • palabras que empiezan, terminan o contienen determinados caracteres
  • %[wordForm|formas léxicas]%, %[tag|etiquetas]%, %[lemma|lemas]% y otros %[attribute|atributos]%
o una combinación de las tres opciones anteriores.
En la lista de palabras se pueden mostrar tres medidas de frecuencia diferentes: %[frequency|frecuencia]%, %[relfreq|frecuencia por millón]% y ARF.
  • ¿Cómo funciona la lista de palabras?
    La lista de palabras funciona al nivel %[token]%. La configuración por defecto producirá una lista de palabras porque las no-palabras se excluyen automáticamente.

    La lista de palabras puede limitarse por la frecuencia con el ajuste de frecuencia mínima y máxima.

    Se pueden obtener frecuencias para una lista de palabras concretas. Utiliza la opción de esta lista: en la pestaña advanzada e introduce los elementos cuyas frecuencias deben calcularse a partir del corpus seleccionado.

    Las %[regex|expresiones regulares]% pueden utilizarse para definir criterios complejos para las palabras que deberían ser incluidas en la lista de frecuencia.
  • Incluir más información en el resultado
    La opción Mostrar como permite incluir más atributos en el resultado. Por ejemplo, se puede generar una lista de frecuencias de formas léxicas con lemas mostrados junto a las formas léxicas. Se permiten hasta 3 atributos.

    Limitación
    Cuando se utiliza la opción Mostrar como, primero se genera una concordancia de todos los %[token|tokens]% que coinciden con los criterios y las frecuencias se calculan a partir de esta concordancia. El tamaño de esta concordancia tiene una limitación técnica de 10 millones de filas de concordancia. Por consiguiente, si el corpus es muy grande, es posible que los datos de frecuencia no se calculen a partir de todo el corpus.

    Esta limitación no puede eliminarse. Para obtener frecuencias de todo el corpus, Sketch Engine puede generar la lista previo pago. Contacte con nosotros.
  • Velocidad y tamaño del corpus
    Sketch Engine está diseñado específicamente para manejar grandes corpus con rapidez. Cualquier búsqueda sólo tardará unos segundos en completarse si el tamaño del corpus es inferior a mil millones de palabras. Para corpus de más de mil millones de palabras, puede tardar un poco más. El uso de expresiones regulares complejas en corpus de gran tamaño puede requerir varios minutos de espera.
  • Requisitos para que la lista de palabras funcione bien
    El único requisito es un corpus %[token|tokenizado]%. Los resultados serán más representativos si se generan a partir de un corpus grande. Sin embargo, no se requiere un corpus mínimo para que la lista de palabras funcione.
